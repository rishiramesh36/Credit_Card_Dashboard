# Credit_Card_Dashboard
I developed this comprehensive weekly dashboard to help stakeholders monitor and analyze credit card operations effectively. The dashboard delivers actionable insights into key performance indicators, customer behavior patterns, and financial trends that drive business decisions.
Features I Implemented:

Dynamic KPI Cards: Real-time revenue, transactions, and customer metrics
Trend Visualizations: Line charts showing week-over-week performance
Segmentation Analysis: Customer breakdown by demographics and behavior
Geographic Heatmap: State-level performance visualization
Card Comparison Charts: Product portfolio performance metrics
Interactive Filters: Drill-down capabilities by time period, region, and customer segment
